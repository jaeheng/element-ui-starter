# element-ui-starter

> A Vue.js project with element-ui

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run all tests
npm test
```
> 本项目是用vue-cli自动生成webpack-vue项目，再添加element-ui, sass-loader, vue-router而成。
